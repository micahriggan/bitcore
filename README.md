Bitcore Node
============
_Requirements_:
- Trusted P2P Peer
- MongoDB Server >= v3.4

Checkout the repo

```
git clone git@github.com:bitpay/bitcore.git
git checkout master
npm install
```

# 2) Setup Bitcoin Node

<details>
<summary><b> Example Bitcoin Mainnet Config </b></summary>
```
whitelist=127.0.0.1
txindex=0
listen=1
server=1
irc=1
upnp=1
```
</details>
<br>

## Listen on different ports than default testnet

```
port=20008
rpcport=20009
rpcallowip=127.0.0.1

rpcuser=username
rpcpassword=password
```

## 3) Run Bitcoin node

## 4) Start Bitcore

```
npm run node
```
